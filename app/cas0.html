Ce module e-learning a été conçu pour vous offrir une possibilité de vous entraîner à activer les concepts fondamentaux que vous avez appris. Le module est construit autour de 4 études de cas qui correspondent aux quatre parties du manuel.


Chaque étude de cas est composée de quatre étapes :

  1. Le quiz autocorrigé permet de vérifier que vous maîtrisez les concepts nécessaires pour mener à bien l'étude de cas.
  2. L'énoncé du cas réunit les informations qui constitueront la base de votre réflexion.
  3. Les questions préparatoires vous invitent à identifier clairement la problématique à traiter et à vous munir d'outils d'analyse pertinents.
  4. La dernière étape est composée d'exercices qui sont destinés à vous guider dans votre analyse.
